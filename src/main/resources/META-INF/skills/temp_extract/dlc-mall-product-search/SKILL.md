# DLC商城商品搜索技能

## 概述

此技能用于自动化DLC商城平台(platform.dlchm.cn)的登录和商品搜索流程。由于平台有图片验证码保护，需要人机协作完成登录。

## 工作流程

### 1. 启动浏览器并访问登录页面

```
1. 调用 browser_start 启动浏览器
2. 调用 browser_action 导航到 https://platform.dlchm.cn/login
3. 调用 browser_view 分析页面结构
```

### 2. 填写登录信息

```
登录凭据（默认）：
- 用户名：zhaohuixin
- 密码：zhaohuixin@321

操作步骤：
1. 在用户名输入框填写用户名
2. 在密码输入框填写密码
3. 点击登录按钮
```

### 3. 处理图片验证码

**重要：验证码需要用户协助完成**

验证码类型：图片选择验证码
- 平台会显示6张图片
- 提示用户选择特定类别的图片（如"戒指"、"鞋类"等）
- AI无法直接识别图片内容

处理流程：
1. 调用 browser_view 获取验证码信息
2. 告知用户验证码提示内容
3. 请用户在浏览器中手动完成验证码选择
4. 用户确认完成后继续后续操作

### 4. 搜索商品

登录成功后：
1. 调用 browser_view 分析页面，找到商品搜索功能
2. 在搜索输入框输入关键词
3. 等待 Elasticsearch 联想下拉框出现
4. 点击联想框中的任意一个选项进行搜索
5. 调用 browser_view 获取搜索结果

### 5. 提取商品信息

从搜索结果页面提取：
- 商品名称
- 商品数量
- 其他相关信息（价格、库存等，如需要）

## 使用示例

**示例1：搜索高压锅**
```
用户：打开DLC商城搜索高压锅，告诉我前5条商品名称

执行步骤：
1. 启动浏览器，访问登录页面
2. 填写用户名和密码
3. 提示用户完成验证码
4. 用户确认后，搜索"高压锅"
5. 提取商品名称列表
```

**示例2：搜索其他商品**
```
用户：在DLC商城搜索电饭煲

执行步骤：
同上，搜索关键词改为"电饭煲"
```

## 注意事项

1. **验证码处理**：图片验证码必须由用户手动完成，AI只能提示用户操作
2. **登录状态**：浏览器会保持登录状态，后续操作无需重复登录
3. **搜索联想**：输入关键词后需要等待并点击联想下拉框的选项
4. **结果数量**：搜索结果可能少于请求的数量，需如实报告

## 页面元素参考

### 登录页面
- 用户名输入框：input[type="text"] 或类似选择器
- 密码输入框：input[type="password"]
- 登录按钮：button[type="submit"] 或包含"登录"文本的按钮

### 搜索功能
- 搜索输入框：商品名称输入框
- 联想下拉框：Elasticsearch 自动补全下拉列表
- 搜索结果：商品列表区域

## 错误处理

1. **验证码失败**：提示用户重新选择验证码图片
2. **登录超时**：刷新页面重新尝试
3. **搜索无结果**：告知用户并建议更换关键词
4. **网络错误**：检查网络连接后重试

## 输出格式

搜索结果应包含：
```
搜索关键词：[关键词]
找到 [N] 条商品：

1. [商品名称1]
2. [商品名称2]
3. [商品名称3]
...
```
